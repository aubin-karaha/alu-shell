# Shell I/O Redirections and Filters

Scripts covering I/O redirection and filters on Ubuntu 20.04 LTS.

## Scripts

- **0-hello_world** - Prints "Hello, World"
- **1-confused_smiley** - Displays a confused smiley "(Ôo)'
- **2-hellofile** - Displays content of /etc/passwd
- **3-twofiles** - Displays content of /etc/passwd and /etc/hosts
- **4-lastlines** - Displays last 10 lines of /etc/passwd
- **5-firstlines** - Displays first 10 lines of /etc/passwd
- **6-third_line** - Displays third line of file iacta
- **7-file** - Creates a file with a tricky name containing "Best School"
- **8-cwd_state** - Saves ls -la output to ls_cwd_content
- **9-duplicate_last_line** - Duplicates last line of file iacta
- **10-no_more_js** - Deletes all .js files in current directory and subdirectories
- **11-directories** - Counts directories and sub-directories in current directory
- **12-newest_files** - Displays 10 newest files in current directory
- **13-unique** - Prints words that appear exactly once from input
- **14-findthatword** - Displays lines containing "root" from /etc/passwd
- **15-countthatword** - Counts lines containing "bin" in /etc/passwd
- **16-whatsnext** - Displays lines with "root" and 3 lines after in /etc/passwd
- **17-hidethisword** - Displays lines not containing "bin" in /etc/passwd
- **18-letteronly** - Displays lines starting with a letter in /etc/ssh/sshd_config
- **19-AZ** - Replaces A with Z and c with e from input
- **20-hiago** - Removes all c and C letters from input
- **21-reverse** - Reverses input
- **22-users_and_homes** - Displays users and home directories sorted by username
- **23-empty_casks** - Finds all empty files and directories
- **24-gifs** - Lists all .gif files without extension, sorted case-insensitively
- **25-acrostic** - Decodes acrostics using first letter of each line
- **26-the_biggest_fan** - Displays top 11 hosts from web server logs
