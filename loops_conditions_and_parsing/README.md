# Shell Loops, Conditions and Parsing

Scripts covering loops, conditions and parsing on Ubuntu 20.04 LTS.

## Scripts

- **1-for_best_school** - Displays Best School 10 times using a for loop
- **2-while_best_school** - Displays Best School 10 times using a while loop
- **3-until_best_school** - Displays Best School 10 times using an until loop
- **4-if_9_say_hi** - Displays Best School 10 times, prints Hi on 9th iteration
- **5-4_bad_luck_8_is_your_chance** - Loops 1-10 with bad/good luck messages
- **6-superstitious_numbers** - Displays 1-20 with superstitious messages
- **7-clock** - Displays time for 12 hours and 59 minutes
- **8-for_ls** - Lists current directory, showing only part after first dash
- **9-to_file_or_not_to_file** - Gives information about the school file
- **10-fizzbuzz** - Displays FizzBuzz from 1 to 100
- **11-read_and_cut** - Displays username, uid, and home from /etc/passwd
- **12-tell_the_story_of_passwd** - Displays /etc/passwd content as a story
- **13-lets_parse_apache_logs** - Displays visitor IP and HTTP status from Apache log
- **14-dig_the-data** - Groups visitors by IP and status code, sorted by occurrences
