# Shell Basics

Scripts covering shell navigation and file manipulation on Ubuntu 20.04 LTS.

## Scripts

- **0-current_working_directory** - Prints the absolute path of the current working directory
- **1-listit** - Displays contents of the current directory
- **2-bring_me_home** - Changes working directory to the user's home directory
- **3-listfiles** - Lists current directory contents in long format
- **4-listmorefiles** - Lists contents including hidden files in long format
- **5-listfilesdigitonly** - Lists contents with numeric user/group IDs including hidden files
- **6-firstdirectory** - Creates a directory named my_first_directory in /tmp/
- **7-movethatfile** - Moves the file betty to /tmp/my_first_directory
- **8-firstdelete** - Deletes the file betty from /tmp/my_first_directory
- **9-firstdirdeletion** - Deletes the directory /tmp/my_first_directory
- **10-back** - Changes working directory to the previous one
- **11-lists** - Lists files in current, parent, and /boot directories in long format
- **12-file_type** - Prints the type of the file /tmp/iamafile
- **13-symbolic_link** - Creates a symbolic link to /bin/ls named __ls__
- **14-copy_html** - Copies newer or non-existing HTML files to the parent directory
- **15-lets_move** - Moves files beginning with an uppercase letter to /tmp/u
- **16-clean_emacs** - Deletes all files ending with ~ in the current directory
- **17-tree** - Creates the directory structure welcome/to/school
